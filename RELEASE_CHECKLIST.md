## 1. アプリ動作

- [ ] Android Studio でエミュレータ起動後、アプリがクラッシュせず開く
- [ ] 実機でアプリがクラッシュせず開く
- [ ] 新しい場面を追加できる
- [ ] 場面名を編集できる
- [ ] カードに入力できる

## 6. Data safety の想定

現時点の実装では、入力内容は端末内の localStorage に保存され、外部サーバーへ送信しません。

- データ収集: なし
- データ共有: なし
- アカウント作成: なし
- 位置情報: 使用しない
- 連絡先: 使用しない
- 広告 ID: 使用しない
- 外部分析 SDK: 使用しない
- クラウド同期: なし

将来、広告、分析、クラウド同期、ログイン、通知などを追加した場合は、この内容を必ず見直してください。
